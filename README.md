# Evaluating batters in cricket (Markov chain & survival analysis)

## Shiny dashboard (Python)

1. Open a terminal in this project folder.
2. Install the Python packages: `pip install -r requirements.txt`
3. Start the app: `python -m shiny run app.py`
4. Open the address shown in the terminal (usually `http://127.0.0.1:8000`) in your browser.

## R Markdown scripts

The `Rmd_scripts/` notebooks need **R** and the packages listed at the bottom of `requirements.txt`. Install them in R with `install.packages(...)` as shown there.
